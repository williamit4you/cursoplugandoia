# Sales Page Analytics Spec

Este pacote segue uma abordagem de Spec Driven Development para a vertical de analytics da pagina de vendas.

Arquivos:
- `00-visao-geral.md`: objetivos, escopo, funil e criterios de sucesso
- `01-modelo-dados.md`: tabelas, enums e decisoes de modelagem
- `02-tracking-e-apis.md`: eventos, payloads, ingestao e consultas
- `03-dashboard-admin.md`: UX, KPIs, filtros e comportamentos da tela admin
- `04-plano-implementacao.md`: fases sugeridas para entrega incremental

Objetivo central:
- manter o Meta Pixel funcionando para campanhas;
- adicionar analytics proprietario para a landing page;
- criar um dashboard administrativo com metricas reais do funil.
