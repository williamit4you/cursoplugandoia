# SDD-019: Perguntas cria Vídeos Posts & Click Analytics View

## 1. Objective
Define the UI layout to display social publications and click metrics specifically for Q&A videos.

---

## 2. Page Specifications
* **Q&A Posts Table**:
  * Displays: Question content, generated video preview, platform destinations.
  * Shortlink column containing `/p/[socialPostId]`.
* **Click Metrics Card**: Tracks CTA clicks, user agents, referrers, and conversion rates.
