# SDD-011: Vídeo com Código Posts & Analytics View

## 1. Objective
Specify the post management screen for code videos.

---

## 2. Page Specifications
* **Post Table**:
  * Displays related `SocialPost` rows with social destinations.
  * Integration link builder containing `/p/[socialPostId]`.
* **Clicks Tracker Card**: Displays views, clicks, and a dynamic graph showing engagement trends over time.
