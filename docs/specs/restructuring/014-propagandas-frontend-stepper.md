# SDD-014: Criar Propaganda Stepper Component

## 1. Objective
Design the user interface for running product ad generation pipelines.

---

## 2. Interactive Steps
* **Ad Configuration Page**: Form for product link inputs, discounts, CTA labels, aspect ratios, and styles (funny, journalism, ironic).
* **Generation Stepper Layout**:
  1. *Composição Comercial* (Copyscript generation)
  2. *Locução comercial* (Vocal synthesis)
  3. *Renderização* (Ad video creation)
* **Logs Panel**: Reusable real-time log reader component.
