# SDD-018: Perguntas cria Vídeos Stepper UI Component

## 1. Objective
Define the user interface layout for the Q&A stepper running view.

---

## 2. Interactive Steps
* **Q&A Input**: Input field for question text and checkbox to choose between custom overlays or Pexels media.
* **Generation Stepper Layout**:
  1. *Análise Filosófica / Explicativa* (Generates script)
  2. *Locução Explicativa* (TTS rendering)
  3. *Renderização de Vídeo* (MP4 composition)
* **Logs Panel**: Reusable real-time log reader component.
