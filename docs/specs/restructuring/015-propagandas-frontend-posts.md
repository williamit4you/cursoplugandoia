# SDD-015: Criar Propaganda Posts & Click Analytics View

## 1. Objective
Define the UI layout to display social publications and click metrics specifically for product ads.

---

## 2. Page Specifications
* **Ads Table View**:
  * Displays: Product thumbnail, title, target affiliate link.
  * Shortlink column containing `/p/[socialPostId]`.
* **Click Metrics Card**: Tracks CTA clicks, user agents, referrers, and conversion rates.
