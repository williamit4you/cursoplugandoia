# Shopee Video Pipeline Specs

Esta pasta contem a especificacao orientada a desenvolvimento do pipeline de videos da Shopee.

Ordem recomendada de leitura:

1. `00-visao-geral.md`
2. `01-modelo-dados.md`
3. `02-estados-e-transicoes.md`
4. `03-orquestrador.md`
5. `04-cron-e-timer.md`
6. `05-runpod-audio-video.md`
7. `06-merge-afiliado-story.md`
8. `07-vitrine-link-bio.md`
9. `08-dashboard-logs-ux.md`
10. `09-politica-retry-observabilidade.md`
11. `10-plano-implementacao.md`
12. `11-checklist-ia.md`
13. `12-comfyui-workflows.md`

Objetivo da pasta:

- alinhar arquitetura antes do codigo;
- permitir que IAs e devs saibam o que ja existe;
- dividir a implementacao por fases;
- impedir perda de rastreabilidade;
- proteger custo do RunPod;
- padronizar o uso do ComfyUI no POD;
- criar uma tela operacional clara.
